# Static Assets

Use this folder for app-owned static files that Angular should copy into the web build.

- `logos/`: brand marks and app logos.
- `icons/`: icons used by pages, install prompts, or integrations.
- `images/`: photos, illustrations, and other page media.
- `documents/`: downloadable PDFs or public documents.

Keep browser favicon files at `public/favicon.ico` when they must be served from `/favicon.ico`.
